1. Make sure the hostname value in Client.java is set to your computer's hostname value 
	(check using hostname in Mac or ipconfig /all in Windows)

2. Go to directory
	rockykevinleon/eece411/assign2/src

2. Compile the code using
	javac -cp . com/matei/eece411/A2/*.java

3. Generate the stub using (note: we only need to generate the stub for ChatImpl since we only bind that to the rmiregistry)
	rmic -classpath . com.matei.eece411.A2.ChatImpl

4. (OPTIONAL: We do this in the code now, but in case it doesn't work you can do this) Start the rmiregistry process using
	rmiregistry &

5. Run the client/server with the policy file argument using
	java -cp . -Djava.security.policy=<path to policy file> <client/server class name>
	
	For example:
	java -cp . -Djava.security.policy=com/matei/eece411/A2/allAccess.policy com.matei.eece411.A2.ChatImpl <chatroomName>
	java -cp . -Djava.security.policy=com/matei/eece411/A2/allAccess.policy com.matei.eece411.A2.Client <chatroomName> <clientName>
